# Macrocelda-small-cells-en-una-zona-de-alta-demanda
 Modernizar el caso macro/micro hacia un escenario de calle comercial, festival o estadio perimetral donde la capacidad manda tanto como la cobertura. Escenario: Un area urbana funciona razonablemente con una macro LTE, pero en franjas de maxima ocupacion aparecen zonas congestionadas. Se estudia el impacto de anadir una o dos small cells.
